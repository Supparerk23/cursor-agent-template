# Frontend — context

## Scope

- Components, views, styles, client-side logic, UX flows.
- Do **not** own: API contracts, server-side behavior — defer to backend.

## File patterns

- UI components: `**/*.tsx`, `**/*.jsx`, `**/*.vue`, `**/*.svelte`
- Styles: `**/*.css`, `**/*.scss`, `**/*.module.css`
- Client entry and app shell: `src/App.*`, `src/main.*`, `src/pages/**`, `src/components/**`
